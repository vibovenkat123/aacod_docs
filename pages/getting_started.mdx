# Getting started

> These docs assume you use Typescript


## Setup script
If you are on a unix system, you can use the following script to set up a new AACOD project.
It installs node, and initializes a new npm project with AACOD as a dependency in the current directory.

```bash copy
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/vibovenkat123/aacod/main/setup.sh)"
```

## Prereqs

* Node
* Typescript
* Npm, pnpm, or yarn

## Setup
